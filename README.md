# Binkbe Dashboard v1.0.2

Dashboard web para Binkbe — versión 1.0.2.

## Funcionalidades

- **Resumen**: métricas clave (usuarios, ingresos, pedidos, conversión), gráficas de actividad mensual y distribución, tabla de actividad reciente.
- **Analíticas**: visitas totales, tiempo promedio y tasa de rebote con gráfica diaria.
- **Usuarios**: listado de usuarios con rol y estado.
- **Configuración**: ajustes generales de la aplicación.

## Uso

Abrir `index.html` directamente en el navegador, o servir con:

```bash
npm start
```
